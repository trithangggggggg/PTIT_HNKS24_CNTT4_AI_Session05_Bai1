Phương án B là lựa chọn tối ưu nhất vì chứa đầy đủ 5 thành phần của một prompt tốt.

Vai trò: AI được định hướng đóng vai Java Senior Developer.
Mục tiêu: Tái cấu trúc logic if-else lồng nhau thành các guard clauses (return sớm).
Ngữ cảnh: Áp dụng cho class DiscountService với logic rẽ nhánh phức tạp.
Ràng buộc: Giữ nguyên nghiệp vụ cũ, không thay đổi kiểu dữ liệu đầu vào/đầu ra và sử dụng Java 11.
Định dạng: Yêu cầu trả về mã nguồn Java hoàn chỉnh kèm giải thích ngắn bằng tiếng Việt.

Nhờ có đầy đủ các thành phần trên, AI được định hướng rõ ràng, giảm nguy cơ làm thay đổi logic tính chiết khấu và tạo ra kết quả dễ sử dụng.
